About me

Ever since I was a young teenager, I was fascinated with video games, how they were built, and began teaching myself how to build my own. This led me to the world of programming, where I was amazed by how powerful simple Python scripts could be and I was captivated by being able to exercise my creativity by building my own projects. Currently, I'm a software engineer specializing in React, Redux, and Python. My main project right now is a full-stack clone of Slack that utilizes Socket IO for messaging functionality.

Median

Median is a full-stack application clone of Medium. It is an online platform where readers can find diverse stories, ideas and can share their prospectives on any topic. Our application incorporating React and Redux on the frontend with Flask, Python and SQLAlchemy on the backend. Live site

Feinbnb

Feinbnb is a full-stack application cloning Aribnb. This project utilizies the following technologies: HTML, CSS, JS (React, Redux, Express, Sequelize), and SQL. Live site

Transmit

Transmit is a full-stack application cloning Slack. This project includes Slack's website (splash page, login, sign up) as well as their desktop application for messaging, channels, and workspaces. Transmit utilizes the following technologies: HTML, CSS, JavaScript, Python, SQL, and Socket.io for messaging functionality. Live site
